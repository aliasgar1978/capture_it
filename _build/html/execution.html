Execution Steps

Execution Steps - High Level

  1. Import project module

  2. Provide auth para

  3. List eligible devices

  4. List commands to be captured

  5. provide output path

  6. execute

Execution Steps - Detailed

  1. Import the necessary module:

    from capture_it import capture

  2. Authentication Parameters:

    auth = {
        'un':'provide username' ,
        'pw':'provide login password',
        'en':'provide enable password'  }

  3. List of devices:

    devices = [
        '192.168.1.1',
        '10.10.10.1',
        # ... list down all ip addresses for which output to be captured  ]

  4. Output path:

    op_path = './captures/'

  5. Commands to be captured:

    CISCO_IOS_CMDS =        [
        'sh run',
        'sh int status',
        'sh lldp nei',
        # .. edit as need  ]
JUNIPER_JUNOS_CMDS = [
        # 'show configuration | no-more',
        'show lldp neighbors | no-more',
        'show interfaces descriptions | no-more',
        # .. edit as need ]
cmds = {
        'cisco_ios'  : CISCO_IOS_CMDS,
        'juniper_junos': JUNIPER_JUNOS_CMDS, }

  6. Start Capturing:

    capture(devices, auth, cmds, op_path, cumulative='Both')

Watch out for the terminal if any errors and see your output in given output path.
